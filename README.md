# E-Commerce Customer Behavior Analytics Using Big Data

## Project Overview
This project analyzes large-scale e-commerce transaction data using Apache Spark (PySpark).

## Technologies Used
- Apache Spark
- Python
- Google Colab
- Pandas
- Matplotlib

## Key Features
- Data Cleaning & Preprocessing
- Revenue Analysis
- Product Performance Analysis
- RFM Customer Segmentation
- Business Insights & Visualization

## Key Results
- Total Revenue ≈ 8.9 Million
- Champions generated the highest revenue
- UK is the dominant market
- Revenue follows Pareto distribution
